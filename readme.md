## 清华日报

学生健康和出行情况报告每日自动提交




### 正常使用说明

* pip install -r requirements.txt
* 配置 conf.ini 输入清华info 用户名与密码
* python report.py
* 创建定时任务 Windows/Linux
> [Windows 创建定时任务](https://www.cnblogs.com/wensiyang0916/p/5773828.html)
2020.1.15 可用
